אוספים

מסע חזותי בין קטגוריות נבחרות ותמות מעודנות

כל אוסף נבנה בקפידה כדי לספר סיפור ייחודי. אנו מזמינים אתכם לגלות את הקטגוריות השונות, להתעמק בתמות המרכזיות ולמצוא את הפריטים המדויקים שמתאימים לחזון שלכם. כל בחירה משקפת מחויבות לאיכות, אסתטיקה ומקוריות.

אוספי מורשת

פריטים עתיקים ומסורתיים שמשלבים היסטוריה עם עיצוב עכשווי. כל יצירה נבחרה בקפידה כדי להעניק תחושה של זמן, מקום ומשמעות תרבותית עמוקה.

עיצוב עכשווי

קולקציה של יצירות מודרניות שמשקפות מגמות עכשוויות, חדשנות ויצירתיות. כאן תמצאו פריטים נועזים ומעוררי השראה שמתאימים למרחבים דינמיים ומודרניים.

טבע ונוף

אוסף המוקדש לאלמנטים אורגניים, חומרים טבעיים וצבעים שנלקחו ישירות מהטבע. פריטים אלו מביאים את השלווה והאותנטיות של הסביבה הטבעית אל תוך הבית.

אומנות מקומית

יצירות של אמנים מקומיים שמשלבים טכניקות מסורתיות עם ביטוי אישי ייחודי. כל פריט מספר סיפור של מלאכה, תרבות וקהילה.

פריטי אספנות

אוסף נדיר של פריטים בעלי ערך היסטורי ואסתטי יוצא דופן. עבור אוהבי האומנות והייחודיות, זהו המקום למצוא יצירות שאי אפשר למצוא בשום מקום אחר.

עיצוב בר קיימא

פריטים שנוצרו תוך מחשבה על הסביבה, באמצעות חומרים ממוחזרים, טכניקות ידידותיות לסביבה ותהליכי ייצור אתיים. יופי שמשלב אחריות.

אוסף הקפסולה העונתי

מבחר מוצרים נבחרים שנאספו בקפידה לעונת החורף, משלבים איכות פרמיום עם עיצוב מודרני ונצחי.

SET 01

מהדורה מוגבלת

פריטים ייחודיים בכמות מוגבלת, מעוצבים במיוחד לאספנים ולחובבי איכות.

SET 02

סדרת האומנים

כל פריט נוצר בעבודת יד על ידי אומנים מקומיים, עם תשומת לב לכל פרט.

SET 03

קולקציית הרטרו

פריטים בהשראת העבר, משלבים נוסטלגיה עם פונקציונליות עכשווית.

SET 04

חזון עכשווי

עיצוב מודרני וחדשני המשקף את רוח הזמן ומציע פתרונות חכמים.

SET 05

סדרת הטבע

מוצרים ידידותיים לסביבה העשויים מחומרים טבעיים ומתחדשים.

SET 06

מהדורת יוקרה

פריטים יוקרתיים עם גימורים מיוחדים וחומרים נדירים.

SET 07

פלטת הצבעים

מגוון עשיר של גוונים ודוגמאות המעוררים שמחה ויצירתיות.

SET 08

סגנון חיים

מוצרים המשתלבים באופן טבעי בשגרת היום-יום ומעשירים את חוויית החיים.

SET 09

שחור לבן

עיצוב קלאסי בגוונים ניטרליים, נצחי ואלגנטי לכל סגנון.

SET 10

משחק טקסטורות

שילוב של משטחים ומרקמים מגוונים היוצרים עומק ועניין ויזואלי.

SET 11

אוסף העונה

מוצרים חדשים המשקפים את רוח העונה הנוכחית ואת המגמות העדכניות.

SET 12

איך עובדת האוצרות

כל פריט בקטלוג שלנו עובר תהליך אוצרות קפדני שמתחיל בחיפוש אחר יצירות ייחודיות ומסתיים בהצגה מדויקת. אנחנו לא רק אוספים פריטים – אנחנו בוחרים סיפורים, מחפשים איכות ומתחייבים לאותנטיות מלאה.

העבודה שלנו מבוססת על עקרונות ברורים: כל פריט חייב לספר משהו, להיות בעל ערך אמנותי או היסטורי, ולהתאים לסטנדרטים הגבוהים שלנו. אנחנו עובדים עם אמנים, אספנים וספקים מהימנים כדי להבטיח שכל דבר שמגיע אליכם הוא אמיתי ומיוחד.

התוצאה היא קטלוג שמשלב יופי, תוכן ומשמעות – מקום שבו כל פריט הוא לא רק מוצר אלא חוויה.

מיון ובחירה

אנחנו בוחנים מאות פריטים ובוחרים רק את אלה שעומדים בקריטריונים שלנו: איכות, ייחודיות ורלוונטיות. כל פריט נבדק לעומק לפני שהוא נכנס לקטלוג.

תיעוד ומחקר

אנחנו חוקרים את הרקע של כל פריט, מצלמים אותו בתנאים מקצועיים וכותבים תיאור מפורט שכולל את הסיפור, הטכניקה והקשר התרבותי שלו.

הצגה והנגשה

הפריט מועלה לקטלוג עם כל המידע הנדרש, תמונות באיכות גבוהה ומחיר שקוף. אנחנו מוודאים שכל לקוח יכול להבין בדיוק מה הוא מקבל ולמה זה שווה.

רגעי אוסף מיוחדים

כל פריט באוסף שלנו מספר סיפור ייחודי, משלב בין מסורת עתיקה לעיצוב עכשווי ומזמין אתכם לגלות את היופי שבפרטים הקטנים.

השראה ממקורות טבעיים

חומרים אותנטיים ועבודת יד מסורתית שמשקפים את התרבות העשירה והמורשת של אזור אגם אטיטלן.

מעוניינים בהדרכה או יש לכם שאלות?

שתפו אותנו בפרטים על מה שמעניין אתכם ונחזור אליכם בהקדם.
